# HangulFlash - Smart Korean Flashcards

A powerful flashcard application designed for Korean language learning with Spaced Repetition System (SRS).

## Features

- **Spaced Repetition System (SRS)**: Uses the SM-2 algorithm to optimize review intervals and maximize retention.
- **Progress Tracking**: Dashboard with stats on total cards, due cards, learned cards, and retention rate.
- **Study Mode**: Interactive flashcards with flip animation and quality ratings (Again, Hard, Good, Easy).
- **Deck Management**: Add, edit, and delete flashcards.
- **Pre-loaded Content**: Comes with a starter set of essential Korean vocabulary.
- **Pronunciation & Examples**: Fields for Romanization/IPA and example sentences.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion (Animations), Lucide React (Icons)
- **State Management**: Zustand (with Local Storage persistence)
- **Routing**: React Router DOM
- **Utilities**: date-fns, clsx, tailwind-merge

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.
